Review

  • HTML describes the structure of the page
  • HTML code is made up of text in angled brackets called elements
  • Each element tells the browser something about the information that sits between opening and closing tags
  • Syntax is the grammatical structure of words and phrases to create coherent sentences
  • Tag or Tag element refers to the HTML codes that define the element in an file such as headings
